API Basejump: Image Search Abstraction Layer

    1) I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
    2) I can paginate through the responses by adding a ?offset=<number> parameter to the URL.
    3) I can get a list of the most recently submitted search strings.

Example Image Search

https://easy-bing-search.herokuapp.com/api/imagesearch?query=funny%20cats&top=1

You can also use offset=<number> to skip the first results

Example History lookup

https://easy-bing-search.herokuapp.com/api/latest/imagesearch?limit=1

Default limit is the 10 latest entries. Max limit is 50
